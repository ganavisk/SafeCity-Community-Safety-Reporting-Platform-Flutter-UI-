# SafeCity – Community Safety App

## Overview
SafeCity is a mobile UI prototype for reporting and monitoring safety incidents within a city.

## Features
- Incident reporting
- Safety alert dashboard
- Community participation
- Simple mobile UI

## Technologies
- Flutter
- Dart
- Material UI

## Run Project

flutter pub get

flutter run
